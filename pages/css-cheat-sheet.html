CSS Cheat Sheet - Properties, Values, and Selectors

color: red; Sets the text color to red.
font-size: 16px; Specifies the size of the font.
background-color: yellow; Defines the background color of an element.
text-align: center; Aligns text content within its container.
display: flex; Defines a flex container for flex items.
position: relative; Specifies the positioning method of an element.
z-index: 1; Determines the stack order of an element.
opacity: 0.5; Sets the opacity level for an element.
box-shadow: 5px 5px 5px #888888; Adds shadow effects to an element.
transform: rotate(45deg); Applies transformations to an element.
transition: all 0.3s ease; Creates smooth transitions between property values.
overflow: hidden; Specifies how content should behave when it overflows its box.
cursor: pointer; Sets the type of cursor to be displayed.

CSS Selectors

.class-selector Selects elements with a specific class.
#id-selector Selects an element with a specific ID.
element-selector Selects all elements of a specific type.
element, element Selects multiple elements.
element, element Selects multiple elements.
element element Selects an element inside another element.
element > element Selects direct children of an element.
:hover Styles an element on mouse hover.
:nth-child(n) Selects elements based on their order.
[attribute="value"] Selects elements with a specific attribute value.
:not(selector) Selects elements that do not match a specific selector.
:first-child Selects the first child of its parent.
:last-child Selects the last child of its parent.
:nth-of-type(n) Selects elements of a specified type based on their order.
