HTML Cheat Sheet - Single Tags

<!DOCTYPE html> Declares the HTML version to the browser.
<meta charset="UTF-8"> Defines the character encoding for the document.
<title>Page Title</title> Sets the title displayed in the browser's title bar or tab.
<img src="image.jpg" alt="Description"> Inserts an image with a specified source and alternative text.
<a href="https://www.example.com">Link</a> Creates a hyperlink to another web page or resource.
<ul> Defines an unordered list.
<ol> Represents an ordered list.
<li> Denotes a list item within <ul> or <ol>.
<table> Creates a table for data display.
<tr> Defines a table row within <table>.
<td> Represents a table cell within <tr>.
<form> Creates an HTML form for user input.
<input> Used within a form to get user input.
<select> Generates a dropdown list within a form.
<textarea> Creates a multi-line text input within a form.
<header> Represents introductory content in a document.
<footer> Denotes a footer for a document or section.
<nav> Defines a set of navigation links.
<aside> Represents content aside from the main content.
<section> Defines a section in a document.
<article> Represents an independent piece of content.
<main> Specifies the main content of a document.
<figure> Represents self-contained content like images or diagrams.
<figcaption> Provides a caption or description for a <figure> element.
<video> Embeds a video into an HTML document.
